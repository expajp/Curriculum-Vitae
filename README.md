## 強み
* 継続的な学習が習慣づいているため、職務に必要な技術へのキャッチアップはすぐ行えます
  * 2017年7月頃から基本的に毎日コードを書くようにしており、[GitHubのプロフィールページ](https://github.com/expajp)より確認できます
  * 主に[Project Euler](https://projecteuler.net/)をRubyで解いており、2018年9月現在96問解答済みです（参加者83万人中上位1.5%以内）
    * これにより、アルゴリズムの設計と実装が得意です
  * 趣味で学習を進める過程でビットコインの原著論文を英語で通読して社外発表しているため、**ブロックチェーンの基盤技術には十分な理解があります**
* エンジニア組織のマネジメントに強い関心を持っています
  * 「**個々人の能力の和よりも大きな力を発揮できるのが組織であり、少数の天才に頼るモデルより再現性が高い**」という考えによるものです
  * 上記の通り、IPAのプロジェクトマネージャ試験に合格しています
  * 「ピープルウェア」、「ザ・ファシリテーター」、「カイゼン・ジャーニー」など組織マネジメントに関わる本の他、「計画の科学」「マズロー心理学入門」などマネジメント周辺の知識が書かれた本を読んで勉強を進めています
    * 読んだ本は適宜読書メモにまとめています
* プレゼンテーションを得意としています
  * 学生時代に5年間、週2で塾講師として勤めていたことによるものです
  * 登壇歴は後述します

### 2016/04(新卒入社) - 2017/10: 株式会社オロ
職務: ビジネスソリューション事業部 エンジニア

#### [社内案件] 経営戦略会議用資料作成の省力化
2017/03-2017/05

経営戦略会議用資料を作成するためのExcel VBAを実装

* 【担当フェーズ】要件定義、基本設計、詳細設計、プログラミング、導入、運用保守、テスト
  * 海外拠点で作成された現地通貨ベースの会計資料から日本円ベースの連結の会計資料に変換するマクロを作成
  * ヒアリングを通した要件定義から導入まで、プロジェクト管理をほぼ1人で担当
  * 新たな担当者に引き継ぎを行い、進捗管理と依頼者とのコミュニケーションを補助

#### ERPパッケージ: ZAC のカスタマイズ
2016/06-2017/10

Classic ASPで構成されたWebアプリケーションを、既存顧客向けにカスタマイズする

* 【担当フェーズ】詳細設計、プログラミング、単体テスト
  * 売上・販売管理システムのグループ間連携、DB間の連携をメインに設計及び実装
  * 特定の状況で稟議が自動承認されるロジックをDBサーバに実装
  * CSVファイルを出力するロジックと条件抽出のための画面を作成
